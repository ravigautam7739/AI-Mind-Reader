# 🧠 AI Mind Reader (Number Guessing Game)

A simple Python program where the **AI guesses the number you are thinking of** between **1 and 100**.

The program uses a **binary search algorithm** to predict the number in the fewest possible steps.

This is a fun beginner project that demonstrates **logical thinking and algorithm efficiency**.

# 🧠 How It Works

1. The user thinks of a number between **1 and 100**.
2. The AI asks if the number is **greater than a specific value**.
3. Based on the answer, the program adjusts its search range.
4. The process repeats until the number is determined.

This method is called **Binary Search**, which reduces the search space by half each time.
