# 🧠 AI Mind Reader (Number Guessing Game)

This is a simple Python program where the **AI guesses the number you are thinking of** between 1 and 100.

The program uses a **binary search algorithm** to quickly predict your number.

---

## 🚀 How It Works

1. Think of a number between **1 and 100**
2. The AI will ask questions
3. Answer with **y** or **n**
4. The AI will predict your number

---

## ▶️ Run the Program

```bash
python guess_no.py
```
